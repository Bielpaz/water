# Water

Estrutura atual do projeto Water.

## Apps
- `water-reminder-web`: versao web publicada no GitHub/Vercel
- `water-reminder-mobile`: versao mobile com Capacitor, iOS e Android

## Objetivo atual
- Web como portal, login e dashboard
- Mobile como app principal com notificacoes e uso diario
- Conta Google e sincronizacao via Firebase

## Estado atual
- Login Google funcionando na web
- Login Google funcionando no mobile
- Sincronizacao via Firestore funcionando
- Web publica com botao para baixar APK Android
- Mobile iOS funcionando localmente
- APK Android gerado para testes

## Direcao de produto
- Hero, Configuracao e Progresso de hoje: mobile como referencia
- Analytics: web como referencia
- Futuro: organizar a base e depois migrar para monorepo
